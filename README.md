# Asesor-de-Gandalf-Stellaris-ESPANOL-CASTELLANO
Asesor para el Stellaris con la voz de Gandalf, en las notificaciones y alguna modificación característica. Es un mod que se puede usar normalmente, "Épico".

# INSTALACIÓN PASO A PASO
1. Descargas el ZIP
2. Descomprimes el ZIP
3. Abres las carpetas, hasta abrir una que se llama "mod"
4. Dentro de ella, verás una carpeta y un archivo.mod, copias ambos en la carpeta de tu Stellaris. En la siguiente ubicación:

C:\Users\Tu Usuario\Documents\Paradox Interactive\Stellaris\mod\Aqui dentro

(Nota, si no tienes la carpeta mod, dentro de Stellaris, la creas, es normal sino has puesto ningún mod antes.)

6. Una vez hayas copiado la carpeta y el archivo.mod dentro de la carpeta \mod.
7. Tendrás que entrar al launcher de Paradox e ir a "Lotes de juego" --> (arriba derecha) "AÑADIR MÁS MODS". Ahí saldrá, lo seleccionas y añades.
8. Y ahora cuando vayas a "Inicio", debajo de jugar, selecciona en "Lote de juego", el lote en el que hayas puesto el mod e inicia el juego.
9. Ahora lo tendrás en asesores, disfrútalo!
